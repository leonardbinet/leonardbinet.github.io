api_etl.utils_secrets module¶

This module contains a function to extract secrets from environment or a secret file.

api_etl.utils_secrets.get_secret(setting, my_secrets={'MONGO_USER': '****', 'MONGO_HOST': '****', 'POSTGRES_DB_NAME': '****', 'AWS_SECRET_ACCESS_KEY': '****', 'AWS_ACCESS_KEY_ID': '****', 'MONGO_PASSWORD': '****', 'POSTGRES_PASSWORD': '****', 'MONGO_DB_NAME': '****', 'API_PASSWORD': '****', 'AWS_DEFAULT_REGION': 'eu-west-1', 'API_USER': '****', 'POSTGRES_HOST': '****', 'POSTGRES_USER': '****'}, env=True)¶

Tries to find secrets either in secret file, or in environment variables. env > secret file Then, set it as environment variable and returns value. :param setting: :param my_secrets: :param env:
